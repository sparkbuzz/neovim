# sparkbuzz/neovim

## Dependencies
The following external dependencies are required:

- eslint
- prettier
- stylua - `brew install stylua` - https://github.com/JohnnyMorganz/StyLua

## For DAP debugger

Install node-debug2

```
git clone https://github.com/microsoft/vscode-node-debug2.git
cd vscode-node-debug2
npm install
gulp build
```
